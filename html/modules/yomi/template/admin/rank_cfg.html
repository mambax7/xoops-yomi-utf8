    【アクセスランキングの環境設定】
    ※アクセスランキングとは外部からのアクセス数を集計したランキングです
    ※このページへのリンクは「rank.php?mode=rev」です

  1. アクセスランキング→ >実施しない | >実施する

  2. ランクインさせる最低アクセス数→

  3. ランキングの集計期間→日ごとに集計する

  4. ランキングの更新頻度(時間)
    ※何時間ごとにランキングを更新するかを決めます
    ※更新頻度は人気ランキングの設定と同じ設定になります。

  5. ランキングデータの最大保持件数→ベスト位まで保存する

  6. ランキング用URLの転送先
    ※通常はサーチエンジンのトップページ()を指定します
    ※トップページの末尾に"?mode=kt"をつけて、"?mode=kt"とすると各アイテムの第一カテゴリトップに転送します
